All Ways to Save

  • Right-click any link/text - "Send to Google Sheet"
  • Click extension icon - saves current page URL
  • Ctrl+I - save selected text or link
  • Ctrl+Shift+I - save current page URL
  • Hover over long links - click the save button
Mac users: Use Cmd instead of Ctrl

Floating Button
